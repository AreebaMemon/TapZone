## How to Run

1. Clone or download the repository.
2. Make sure the `sounds` folder contains the required audio files:
   - `tom-1.mp3`
   - `tom-2.mp3`
   - `tom-3.mp3`
   - `tom-4.mp3`
   - `crash.mp3`
   - `kick-bass.mp3`
   - `snare.mp3`
3. Open `index.html` in a modern web browser.
4. Click any button to hear the corresponding drum sound.
